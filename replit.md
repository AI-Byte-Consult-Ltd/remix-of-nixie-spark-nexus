# AI Byte Consult Website

## Overview
This is the AI Byte Consult company website - a React-based frontend application built with Vite. The company specializes in AI development and business automation, creating scalable solutions for enterprises in hospitality, healthcare, logistics, and research.

## Project Architecture

### Tech Stack
- **Frontend Framework**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS with shadcn/ui components
- **Routing**: React Router DOM v6
- **State Management**: TanStack React Query

### Directory Structure
```
src/
├── App.tsx           # Main application component with routing
├── main.tsx          # Application entry point
├── index.css         # Global styles and Tailwind imports
├── components/       # Reusable UI components
├── pages/            # Page components for routing
├── contexts/         # React context providers
├── hooks/            # Custom React hooks
├── lib/              # Utility functions
└── assets/           # Static assets (images, icons)
```

### Key Pages
- Home
- Ecosystem
- Projects
- Automation
- Contact
- Services

## Development

### Running the Project
The application runs on port 5000 using the Vite development server:
```bash
npm run dev
```

### Building for Production
```bash
npm run build
```
